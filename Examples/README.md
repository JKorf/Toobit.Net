# Examples

| Project | Topic |
|---|---|
| `Toobit.Examples.Api` | Minimal ASP.NET Core API integration, dependency injection, public ticker endpoint, private balances endpoint |
| `Toobit.Examples.Console` | Basic REST ticker request and WebSocket ticker subscription |
| `Toobit.Examples.OrderPlacement` | Spot and futures order placement, order status lookup, and cancellation |
| `Toobit.Examples.OrderBook` | Client-side order book implementation with live console display |
| `Toobit.Examples.Tracker` | Trade tracker with rolling market statistics |

The `ai-friendly` folder contains standalone copy/paste examples intended for quick onboarding and AI coding assistant context.
